# 🏹 弓箭手肉鸽游戏 ⚡

一个现代化的网页版弓箭手肉鸽游戏，采用科技蓝主题设计，具有赛博朋克风格的视觉效果和流畅的操作体验。

## 🎮 游戏特色

### 🎯 核心玩法
- **能量弓箭系统**：角色自动向射程内最近的敌人发射能量箭矢
- **属性成长**：攻击力、攻速、射程、生命值、移速等多维度属性
- **敌人生成**：从屏幕四周随机生成不同强度的敌人
- **道具系统**：击杀敌人有概率掉落蓝色科技道具
- **升级系统**：获得经验值升级，随机提升属性

### 🎨 科技蓝视觉设计
- **赛博朋克UI**：科技蓝配色，发光边框，网格背景
- **能量粒子系统**：升级、击杀、拾取道具时的蓝色能量特效
- **屏幕震动**：受到伤害和击杀敌人时的震动反馈
- **伤害数字**：实时显示造成的伤害数值
- **光环效果**：角色和武器的蓝色能量光环
- **科技背景**：赛博风格的渐变背景和网格纹理

### 🎛️ 操作体验优化
- **双重控制**：支持鼠标和键盘(WASD)双重控制方式
- **智能暂停**：空格键或按钮暂停/继续游戏
- **小地图系统**：实时显示玩家、敌人、道具位置
- **设置面板**：可调节画质、音效、特效等选项
- **升级选择**：升级时可选择想要的属性提升
- **通知系统**：实时提示游戏事件和道具获取
- **性能监控**：FPS显示和对象数量统计
- **波次系统**：渐进式难度提升
- **快捷键**：多种快捷键提升操作效率
- **准星显示**：可选的瞄准辅助功能

## 🎮 操作说明

### 🖱️ 鼠标控制
- **移动鼠标**：角色会自动跟随鼠标移动
- **自动射击**：无需点击，角色会自动攻击范围内的敌人

### ⌨️ 键盘控制
- **WASD / 方向键**：控制角色移动
- **空格键**：暂停/继续游戏
- **ESC键**：打开/关闭设置面板
- **M键**：切换小地图显示
- **C键**：切换准星显示
- **键盘控制优先级高于鼠标**

### 📱 界面功能
- **暂停按钮**：点击右上角暂停按钮
- **设置按钮**：点击右下角设置按钮
- **小地图**：右上角显示游戏区域概览
- **升级选择**：升级时会暂停游戏并显示选择面板
- **通知系统**：右侧显示游戏事件提示
- **性能监控**：显示FPS和对象数量

## 🎯 游戏机制

### 📈 升级系统
- 击杀敌人获得经验值
- 达到经验值上限自动升级
- 升级时随机提升一项属性：
  - 攻击力 +5
  - 攻击速度 +0.2/s
  - 射程 +20
  - 最大生命值 +20（并完全恢复）
  - 移动速度 +0.5

### 👹 敌人系统
- 从屏幕四边随机生成
- 随机血量和大小
- 移动速度和伤害值随机
- 随时间推移生成速度加快

### 💎 道具掉落
- 30%概率掉落道具
- 道具类型随机
- 15秒后自动消失
- 靠近自动拾取
